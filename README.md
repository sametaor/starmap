# Starmap

An opinionated way to access your Github stars! No more hassle with the default and downright ugly Github Stars webpage (unless you like sifting through thousands of pages like me).

> NOTE: This html has been prepared with some usage of AI, expect some bugs or weird formatting issues. Report it as an issue if you do face any!

## How to Use
### Step 1.
Make sure you get a Github Personal Access Token (PAT) so that you don't face issues fetching your starred repos, else you may face a lot of rate limiting
### Step 2.
Ensure your Github PAT has read-only access to your public repositories (if you have starred any repos of your own) as well as to the "Starring" permission below. Once done, save and copy the PAT.
### Step 3.
Enter the Github PAT by pressing the 🔑 button on the top header. We are ready to roll!
