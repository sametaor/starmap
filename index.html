API Rate Limits

Unauthenticated GitHub requests are limited to 60 per hour. Provide a GitHub Personal Access Token (PAT) to increase this limit to 5,000 per hour.

Your token is stored locally in your browser and never sent anywhere except directly to GitHub's API. A fine-grained token with "Public Repositories (read-only)" access is recommended.
